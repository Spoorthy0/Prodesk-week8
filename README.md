# 🎬 CineStream – Netflix Lite Media Discovery App

A simple React-based movie discovery application built as a learning project to explore modern frontend development concepts and API integration.

This project uses the TMDB (The Movie Database) API to display popular movies, search for movies, and manage favorite movies with a smooth user experience.

---

# 🚀 Features

- Browse popular movies from TMDB API
- Search movies dynamically
- Infinite scrolling using Intersection Observer
- Debounced search input for optimized API calls
- Add and remove favorite movies
- Favorites stored using localStorage
- Lazy loading for movie posters
- AI Mood Matcher using Gemini/OpenAI API

---

# 🛠️ Technologies Used

- React.js
- Vite
- Tailwind CSS
- React Router DOM
- TMDB API
- LocalStorage

# ⚙️ Installation

```bash

```

# 🔑 Environment Variables

Create a `.env` file and add:

```env
VITE_TMDB_API_KEY=your_tmdb_api_key
VITE_OPENAI_API_KEY=your_api_key
```
